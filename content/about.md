Hi, I'm the author of this blog. This is a single page (not a blog post) written
in Markdown.

In Hugo, any `.md` file under `content/` becomes a page. The URL is derived from
its path:

- `content/about.md` → `/about/`
- `content/posts/hello-world.md` → `/posts/hello-world/`

## Tools used

- **Hugo** — static site generator written in Go
- A custom theme located at `themes/simple/`
- Plain CSS, no JavaScript
